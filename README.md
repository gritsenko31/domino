## ğŸ® How to Play

### Objective
Be the first player to play all your tiles, or have the lowest total when the game is blocked!


### Game Rules
- Each player starts with 7 tiles
- Match the number of dots on one end of your tile with an end on the board
- First tile can be any piece
- Play tiles on either end of the chain
- If you can't play, draw from the pile
- When pile is empty and you can't play, pass your turn
- First to play all tiles wins
- If game is blocked, lowest total dots wins
